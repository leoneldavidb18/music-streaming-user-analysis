# Music Streaming — User Behavior Analysis (Python)

## Descripción
Proyecto de análisis de datos enfocado en el **comportamiento de usuarios dentro de una plataforma de transmisión de música**.  
El objetivo es explorar, limpiar y analizar los datos para identificar patrones de uso, preferencias y métricas relevantes que apoyen la toma de decisiones de producto.

## Objetivos
- Analizar el comportamiento de escucha de los usuarios.
- Identificar patrones de consumo de música.
- Evaluar métricas clave relacionadas con actividad y preferencias.
- Aplicar limpieza y transformación de datos para garantizar consistencia.

## Dataset
El dataset contiene información relacionada con la actividad de los usuarios en una plataforma de streaming musical, incluyendo variables asociadas al consumo de contenido y características del usuario.

*(Dataset simulado con fines académicos)*

## Principales tareas realizadas
### 1) Limpieza y preparación de datos
- Normalización de variables.
- Manejo de formatos y tipos de datos.
- Verificación de valores inconsistentes.

### 2) Análisis exploratorio (EDA)
- Evaluación de patrones de uso de la plataforma.
- Identificación de tendencias en el consumo musical.
- Análisis de métricas relevantes para el negocio.

### 3) Análisis de comportamiento de usuarios
- Segmentación básica de usuarios según su actividad.
- Comparación de patrones de escucha.
- Extracción de insights a partir de los datos.

## Resultados clave
- Identificación de patrones de consumo y preferencias de los usuarios.
- Hallazgos relevantes para la optimización de la experiencia del usuario.
- Base de análisis lista para futuras visualizaciones o modelos predictivos.

## Tecnologías y habilidades
- Python
- Pandas
- Análisis Exploratorio de Datos (EDA)
- Limpieza de Datos
- Estadística descriptiva
- Visualización de datos
